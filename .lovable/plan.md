

## Страница «Подписка» (`/pricing/`)

### Название и URL
Страница **"Тарифы"** по пути `/pricing/` — в 2026 году "pricing" более универсально и понятно, чем "subscription".

### Подход к UI
**Карточки** (не таблица) — лучший выбор для 2026:
- Адаптивность: на мобильных карточки складываются вертикально без горизонтального скролла
- Визуальное выделение Pro-тарифа (border, badge "Популярный")
- Чёткое разделение: каждая карточка — самодостаточный блок с CTA

### Детали карточек

**Standard (бесплатный):**
- Оповещения в общей очереди
- Базовые фильтры (валюта, площадка)
- Оптимизированный пакет запросов
- Умеренная реклама
- CTA: «Начать бесплатно» → `/login/`

**Pro (платный):**
- Визуально выделен (синий border, badge)
- Приоритетные мгновенные оповещения (Telegram/Push)
- Zero Ads
- Расширенные лимиты
- Все фильтры включая YTM, объём лота, финпоказатели
- CTA: «Подключить Pro» → `/login/`

### Стилистика
- Соответствует дизайн-системе проекта (Manrope, цвета из CSS-переменных)
- Pro-карточка: `border-2 border-blue-500`, badge `bg-blue-500 text-white`
- Standard: `border border-gray-200`
- Иконки-галочки (✓) для включённых функций, крестики или серый текст для ограничений
